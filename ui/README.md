# AGIJobManager UI

## Requirements
- Node >= 20.11.0

## Setup
- `npm ci`
- `cp .env.example .env.local`
- `npm run dev`

## Commands
- `npm run build`
- `npm run lint`
- `npm run test`
- `npm run test:e2e`

## Notes
- Read-only mode works without wallet connection.
- Write operations are simulation-first with error decoding.
