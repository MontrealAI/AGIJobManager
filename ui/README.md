# AGIJobManager UI

## Run
```bash
npm ci
cp .env.example .env.local
npm run dev
```

## Demo mode
Set `NEXT_PUBLIC_DEMO_MODE=1` for deterministic fixtures and read-only showcase.

## Commands
- `npm run lint`
- `npm run typecheck`
- `npm run test`
- `npm run test:e2e`
- `npm run test:a11y`
- `npm run build`
- `npm run docs:versions`
- `npm run docs:check`
- `npm run docs:screenshots`
