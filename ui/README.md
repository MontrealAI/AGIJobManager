# AGIJobManager UI (Next.js)

Institutional dApp + ops console with read-only-first architecture and deterministic Demo Mode.

## Quick start
```bash
npm ci
cp .env.example .env.local
NEXT_PUBLIC_DEMO_MODE=1 npm run dev
```

## Commands
- `npm run lint`
- `npm run typecheck`
- `npm run test`
- `npm run test:e2e`
- `npm run test:a11y`
- `npm run docs:versions`
- `npm run docs:check`
- `npm run docs:screenshots`
