# AGIJobManager UI

Standalone Next.js App Router frontend for institutional dApp + ops workflows.

## Quick start

```bash
npm ci
cp .env.example .env.local
NEXT_PUBLIC_DEMO_MODE=1 npm run dev
```

## Commands

```bash
npm run check:no-binaries
npm run lint
npm run typecheck
npm run test
npm run test:e2e
npm run test:a11y
npm run test:headers
npm run docs:versions
npm run docs:contract
npm run docs:check
npm run build
```
