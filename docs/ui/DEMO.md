# Demo Mode

Set `NEXT_PUBLIC_DEMO_MODE=1` and use `/demo` for deterministic scenarios.

| Scenario | Coverage |
|---|---|
| open | Open jobs |
| assigned | Assigned jobs |
| completion | Completion requested |
| disputed | Dispute lane + settlement paused |
| settled | Settled jobs |
| edge | deleted slots, malformed URI, huge values, degraded RPC |
