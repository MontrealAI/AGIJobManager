# UI Overview

AGIJobManager UI is an institutional-grade dApp + ops console with read-only first behavior, simulation-first write patterns, and deterministic demo mode.

- Routes: `/`, `/jobs`, `/jobs/[jobId]`, `/admin`, `/design`, `/demo`
- Demo mode: `NEXT_PUBLIC_DEMO_MODE=1` with scenario switch (`?scenario=`)
- Text-only visuals: Mermaid + generated SVG plates in `docs/ui/graphics/`
