# UI Overview

Institutional-grade Next.js dApp + ops console for AGIJobManager with read-only-first UX, simulation-first writes, and deterministic demo mode.

## Pages
- `/` dashboard
- `/jobs` jobs ledger + CSV export
- `/jobs/[jobId]` job detail + URI safety controls
- `/admin` owner-only ops console
- `/design` design system gallery
- `/demo` scenario matrix
